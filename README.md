## üåü Overview

Say goodbye to kitchen conversion confusion! This Recipe Unit Converter is a free, modern web application designed to provide instant and accurate conversions for grams, ounces, cups, milliliters, and more. Whether you're a professional chef or a home cook, this tool helps you perfect your recipes with precision and confidence.

The core of this project is its commitment to a clean user experience, powered by a lightweight, dependency-free conversion engine and wrapped in a visually stunning, immersive interface.

## üöÄ Getting Started

To get a local copy up and running, please follow these simple steps.

### Prerequisites

-   Node.js (v16 or later)
-   npm, yarn, or pnpm

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/hsinidev/recipe-unit-converter.git
    cd recipe-unit-converter
    ```

2.  **Install dependencies:**
    ```sh
    # Using npm
    npm install

    # Or using yarn
    yarn install

    # Or using pnpm
    pnpm install
    ```

3.  **Start the development server:**
    ```sh
    # Using npm
    npm run dev

    # Or using yarn
    yarn dev

    # Or using pnpm
    pnpm dev
    ```

4.  Open your browser and navigate to `http://localhost:5173` (or the port specified in your console).
